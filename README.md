# CSS-RGB Colors

A practical and interactive project to understand how **RGB** (Red, Green, Blue) colors work in web design and how to manipulate them using CSS.

## How to use it

1. Clone the repository:
`git clone https://github.com/fabicarrieri/learn-cssrgb-colors.git`

2. Open the `index.html` file in your favorite browser.

3. Experiment with sliders or numeric values ​​in the code.
